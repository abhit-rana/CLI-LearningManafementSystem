This is Command Line Learning Management System(LMS) where the instructor and students have following options availabile in their menu's:

INSTRUCTOR MENU
1. Add class material
2. Add assessments
3. View lecture materials
4. View assessments
5. Grade assessments
6. Close assessment
7. View comments
8. Add comments
9. Logout

STUDENT MENU
1. View lecture materials
2. View assessments
3. Submit assessment
4. View grades
5. View comments
6. Add comments
7. Logout
